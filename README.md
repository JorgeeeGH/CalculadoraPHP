# CALCULADORA EN PHP :heart_eyes:
## En este proyecto vamos a realizar una calculadora básica totalmente funcional, utilizando PHP para el funcionamiento y HTML y CSS para el diseño.

Este proyecto va a ser realizado por mí, pero usando dos cuentas diferentes de GitHub, para simular un ejemplo real de trabajo en grupo usando esta herramienta:
- **JorgeeeGH**
- **Jorggahe**

Este repositorio incluirá todos los archivos necesarios (html, css y php) necesarios para garantizar el correcto funcionamiento de la aplicación web.
Además, contendrá otros archivos como:
- Guías de uso.
- Manual técnico.
- FAQs (preguntas más frecuentes).
- Issues (errores y futuras mejoras).

Los objetivos de aprendizaje que se pretenden conseguir con esta práctica son los siguientes:
- Comprender el trabajo colaborativo o en grupo en proyectos reales.
- Utilizar ramas o *branches*.
- Gestionar *pull requests*.
- Resolver conflictos de código.
- Mejorar el uso del control de versiones.
- Documentar correctamente un proyecto.
